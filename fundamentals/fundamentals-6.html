Control Flow

conditionals, for-loops, while-loops and...just...loops?

If-statements in Rust do not need parentheses around their conditions, like with Java or C:

fn main() {
 let i = 5;
 if i == 0 {
  println!("Zero!"); // Compiles, does not execute
 } else {
  println!("ðŸ˜”");
 }
}

Rust doesn't have ternary operators (that is, single-line conditional operations), but not only can its if-statements be fitted onto one line, the result can immediately be assigned to a variable:

fn main() {
 let i = 5;
 let j = if i > 2 {10} else {0};
 println!("{}", i + j);
}

For-loops in Rust are syntactically similar to that of Python:

fn main() {
 for i in 0..10 { // For-loop with range
  println!("{}", i*i);
 }

 println!("\n");  let arr = vec![5; 10]; // Creates vector with 10 elements, each element being the int 5
 for j in arr { // For-loop with iterable structure, like a vector
  println!("{}", j*j);
 }
}

As can be seen in the above example, ranges in Rust are defined with a double full stop between the lower and upper bounds.
